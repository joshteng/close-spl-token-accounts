# Close all Token Accounts with 0 Balance

Set your environment variables with your private key and RPC. Copy `.env.example` to `.env` then set the values.

```sh
pnpm i
pnpm run keypair
```

or sign via ledger

```sh
pnpm i
pnpm run ledger # connect and unlock your ledger first
```

## Why?
- 1 tx
- fast and easy
